Starting One Game a Month This Year

Posted on February 10, 2014

A little late, but I’ve decided as my New Year’s resolution to start the one game a month challenge again. For those that are not familiar with this challenge, it is where you attempt to create one game for every month of the year. Now, these games are usually very simple, but that’s not really the point. You have to be able to finish a project all on your own. On top of that, you have to be able to keep yourself from trying to do too much. I’m sure it sounds pretty silly, but it is surprisingly difficult.

For my first project, I decided to start my Valentine’s gift to my girlfriend early. Usually I make a program, with some kind of Valentine’s day theme to it. The last couple were pretty quick one-off programs, but this one has a little less fluff to it. I plan to write about it once Valentine’s day has come to pass.
